<h2>프로젝트 내용</h2>

**1. 홈페이지**
<br/>
인테리어, 디지털, 스포츠 등 각 카테고리 별로 쉽게 접근할 수 있는메뉴 제공 + 검색 창 
<br/>
**2. 상품 페이지/상세 정보**
<br/>
제품 설명, 가격 정보 제공<br/>
**3. 사용자 리뷰 및 평점**
<br/>
다른 사용자들이 작성한 리뷰, 평점을 통대 상품에 대한 신뢰도를 높임<br/>
